Application Gate — Private Briefing Model

Families apply via apply.html on the existing site. IB reviews. 20 confirmed seats. Setter contacts every applicant. AO gets paid after seat 20 — not before.

Model

Not a webinar. An invite-only private briefing. Families apply at theivybrothers.com/apply. IB reviews in 24hrs. Accepted get a personal invitation from Jake. Every applicant — accepted or not — gets a setter touchpoint. AO contracted only after 20 seats are confirmed.

Application Form Fields

  • Parent full name + email + phone
  • Student's current school
  • Student's current grade (8–12)
  • Country of origin
  • Country of residence
  • Target schools (up to 3)
  • Strongest current activity or project
  • Biggest concern about the process
  • Worked with a consultant before?
  • How did you hear about Ivy Brothers?

Seat Routing — Soft, Not a Hard Gate

  • Strong fit → invite to current session
  • Good fit, session full → priority waitlist for next run
  • Early grade (8–10th) → Hook Program path, nurture until ready
  • International applicant → flag for AO premium angle + international email
  • Previously used Crimson / Solomon → setter frames IB as the correction
  • Every applicant gets a setter call regardless of routing

03

CRM Options — Simple to Full-Stack

Three tiers: free to get started, mid-range sales-focused, and all-in-one. Recommendation depends on how fast they want to move and what they're willing to pay monthly.

Context

They're already paying for Zapier + Instantly separately. The right CRM eliminates both. At minimum they need: lead capture, setter task queue, call tracking, and automated email sequences. Everything else is a bonus.

Free Start

HubSpot Free

$0/mo to start · Paid plans from $20/mo
  • CRM + pipeline tracking built in
  • Contact records with activity log
  • Email templates + tracking
  • Meeting scheduler (replaces Calendly)
  • Basic automation on free tier
  • Mobile app for setter use
—Email sequences require paid tier ($20+). Automation limited on free. Gets expensive fast if you add features. Not a replacement for Instantly.

Close.com

$49/mo per user
  • Built specifically for sales teams
  • Email sequences native (replaces Instantly)
  • Built-in calling + SMS
  • Activity feed per lead — setter sees everything
  • Pipelines + deal tracking
  • Fast to set up — no heavy onboarding
—No native automation builder (still needs Zapier for complex flows). Calendar booking requires integration. Pricier per seat for a larger setter team.

Attio

Free tier · $34/mo paid
  • Modern UI — easy for non-technical team
  • Flexible data model (custom fields for IB use case)
  • Pipeline views + automations
  • Native integrations with Gmail + Calendar
  • Free tier is genuinely usable
  • Good for small teams moving fast
—Email sequences not as mature as Close or GHL. Outreach still needs a separate tool. Better as a CRM than an all-in-one.
Recommended

GoHighLevel

$97/mo (all-in-one) · $297/mo (agency tier)
  • CRM + pipeline native
  • Email sequences native — replaces Instantly
  • Workflow automation — replaces Zapier
  • Calendar + booking — replaces Calendly
  • SMS outreach built in
  • Forms + landing pages built in
  • Reporting dashboard for setter KPIs
  • One login for the whole stack
—Steeper initial setup than simpler CRMs. UI is dense — takes 1–2 weeks to feel comfortable. Worth it if the full stack is being replaced.

Notion + Make

~$16/mo Notion · Make from $9/mo
  • Lowest learning curve for non-sales teams
  • Notion as a visual CRM database
  • Make replaces Zapier at lower cost
  • Fully customizable to IB's exact fields
  • Easy for setter to log notes per lead
  • Works well as a starter CRM
—Not built for sales. No native email sequences — still needs Instantly or another tool. Manual work increases as volume grows. Not scalable past ~50 leads/week.

Pipedrive

$15/mo per user (Essential) · $29/mo (Advanced)
  • Clean pipeline UI — visual deal stages
  • Email sync + templates
  • Activity reminders for setter follow-ups
  • Automations on Advanced tier
  • Good reporting on close rates + pipeline value
  • Low friction to get started
—Email sequences require Advanced tier or add-on. No SMS native. Still needs Zapier or Make for complex automation flows.
